Welcome to fritz-tools
======================

This is a collection of useful scripts, primarily for use with the bootloader of AVM networking devices.

Supported Devices
-----------------

These devices are confirmed working using the tools provided in this repository.

 - FRITZ!Box 4020
 - FRITZ!Box 4040
 - FRITZ!Box 7312
 - FRITZ!Box 7320
 - FRITZ!Box 7330
 - FRITZ!Box 7330 SL
 - FRITZ!Box 7360 (v1, v2)
 - FRITZ!Box 7360 SL
 - FRITZ!WLAN Repeater 300E
 - FRITZ!WLAN Repeater 450E
 - FRITZ!WLAN Repeater 1750E

Among others, the following models are currently **not** supported:

- FRITZ!Box 7362 SL
